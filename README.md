# Tivexia BPO Website

## Overview
AI-powered BPO, customer service, sales, tech, quality, bookkeeping, payroll, app development, SEO and marketing services for the US, UK, MEA, Europe, AUS and NZ. 24/7 support at the best price on Earth.

## Tech Stack
- **React 19** - Frontend framework
- **Vite 7** - Build tool and dev server
- **Tailwind CSS 4** - Utility-first CSS framework
- **TypeScript** - Type safety
- **React Router** - Client-side routing

## Getting Started

### Prerequisites
- Node.js >= 20.0.0
- npm >= 8.0.0

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
The development server will start at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Project Structure
- `src/` - Source code
- `public/` - Static assets
- `index.html` - Entry HTML file
- `vite.config.ts` - Vite configuration
- `tsconfig.json` - TypeScript configuration

## Features
- Single file build output (via vite-plugin-singlefile)
- Path alias support (`@/` → `src/`)
- Tailwind CSS with type safety
- React Router for page navigation
